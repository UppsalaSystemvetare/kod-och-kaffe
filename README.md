Uppgifter till programmeringsstugan

######1. HighLow
Uppgift: Lös felen i koden för att skapa ett enkelt spel där användaren ska gissa vilket tal programmet "tänker" på.  
Svårighetsgrad (1-5): 1

######2. EncryptDecrypt

Svårighetsgrad (1-5): 2
